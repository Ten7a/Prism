# Prism

Prims will be a simple image generator. It's a token based generator where users can input a prompt, select a model, ratio, quality (from a preselect list like 1k, 2k, 4k) batch count, and they can attach reference images. 

The generated images are saved into their "library". 

The user should be able to login and buy tokens via stripe on web and revenuecat on mobile. Add ads via google ad sense and googles admob on mobile builds.

Write a good privacy policy + terms of service that is good and grdp conform. 

## Technical details
- OpenRouter
- Sveltekit for web
- Capacitor for mobile
